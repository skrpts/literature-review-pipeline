## Purpose

A text-based template for the PRISMA 2020 flow diagram. Used to document the study identification, screening, and inclusion process in a literature review.

## PRISMA 2020 Flow Diagram

```
IDENTIFICATION
├── Records identified from databases (n = )
│   ├── Semantic Scholar (n = )
│   ├── PubMed (n = )
│   ├── Scopus (n = )
│   └── Other databases (n = )
├── Records identified from other sources (n = )
│   ├── Citation searching (n = )
│   ├── Grey literature (n = )
│   └── Expert recommendations (n = )
└── Duplicate records removed (n = )

SCREENING
├── Records screened (n = )
│   └── Records excluded (n = )
│       └── Reasons: [not relevant / wrong population / wrong intervention / etc.]
├── Reports sought for retrieval (n = )
│   └── Reports not retrieved (n = )
│       └── Reasons: [full text unavailable / not in English / etc.]
└── Reports assessed for eligibility (n = )
    └── Reports excluded (n = )
        ├── Wrong study design (n = )
        ├── Wrong population (n = )
        ├── Wrong outcomes (n = )
        └── Other reasons (n = )

INCLUDED
├── Studies included in review (n = )
└── Reports of included studies (n = )
```

## Usage Notes

- Fill in the numbers as the search and screening process progresses
- This template follows the PRISMA 2020 statement (Page et al., 2021)
- For systematic reviews, this diagram is typically required by journals
- For narrative reviews, it demonstrates rigour but may be optional
- Convert to a visual diagram for publication using standard diagramming tools
